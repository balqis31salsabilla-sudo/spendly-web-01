
# Spendly — Responsive Web (Single-File)

Cara deploy (pilih salah satu):
- GitHub Pages: buat repo → upload `index.html` → Settings → Pages → Deploy from a branch (main / root).
- Netlify Drop: ekstrak ZIP, lalu drag & drop folder ini ke https://app.netlify.com/drop
- Vercel: New Project → Import → Other/Static → upload folder yang berisi `index.html`.

Catatan: Aplikasi ini memakai CDN (Tailwind & Chart.js). Pastikan jaringan tidak memblokir CDN.
